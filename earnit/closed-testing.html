EarnIt: Closed Testing Guide

EarnIt for Android · closed test

Thanks for helping test EarnIt! You don't need any technical background. If you can use an Android app, you're qualified. This guide covers how to think about testing and some specific things to try.

How to think about testing

You're not following a strict script. You're using the app the way a real person would, and also trying things a real person might accidentally do.

The goal is to find anything that:

  • Crashes the app or causes an error message
  • Looks broken or out of place (missing text, overlapping buttons, things that don't fit the screen)
  • Doesn't do what you'd expect
  • Feels confusing, even if it technically "works"

That last one is just as important as the others. If something made you stop and think "wait, what does this do?" That's useful feedback.

A few good habits:

  • Go slow and actually read the screens
  • Try tapping things you're not sure about
  • Try going back when you're mid-way through something, then coming back
  • Try leaving the app and coming back (press the home button, do something else, reopen EarnIt)
  • Try rotating your phone sideways on different screens
  • Don't just do the "obvious" path. Try skipping steps, tapping in a different order, leaving fields empty

You won't break anything permanently. If something goes wrong you can always clear the data in Settings → Data & Backup → Clear All.

What to report

If you find something, note:

  1. What you were doing (e.g. "I was adding a new task")
  2. What happened (e.g. "the app closed" or "the button did nothing")
  3. Whether you could make it happen again

A screenshot or screen recording is a huge help if you can manage it. Send feedback to hello@secondmondaystudios.com.

Things to try

Work through these in order. Each one builds on the last. Take your time and explore along the way.

1. Create your first reward

A reward is something you want to earn, like a takeout night or a lazy Sunday.

  1. Open the app. You'll see the Prizes screen.
  2. Create a new reward.
  3. Give it a name and a point cost (try something between 10 and 30 points).
  4. Optionally add a description, something that motivates you.
  5. Scroll down to Tasks and tap Add task.
  6. Create a task, give it a name and save it.
  7. Back on the reward screen, try marking the task as Mandatory.
  8. Save the reward.

Does the reward appear on the home screen? Does it show 0 points toward your target?

2. Log a task

Logging a task earns you points toward a reward.

  1. From the home screen, tap your reward card.
  2. You'll see the reward detail screen with your task listed.
  3. Tap the task to log it. Add a note if you like (optional).
  4. Confirm the log.

Did the points go up? Did the progress bar move? Try logging the same task again. Does it let you, or does it say you've already done it?

3. Claim your reward

Once you've earned enough points (and completed any mandatory tasks), you can claim.

  1. Keep logging tasks until the progress bar is full.
  2. Tap Claim.
  3. Choose either Start over (keep the reward, reset the balance) or Archive (retire it).

Does the reward disappear from the home screen after archiving? Can you find it in History? Try tapping Earn Again on a history entry. Does it bring the reward back?

4. Try the home screen widget

The widget lets you track a reward and log tasks without opening the app.

  1. Long-press an empty area of your Android home screen.
  2. Tap Widgets, find EarnIt, and add it.
  3. When prompted, pick a reward to track.
  4. Optionally change the widget label (this is just the display name on your home screen; it doesn't change the reward name in the app).
  5. Back on the home screen, tap the + LOG button on the widget.
  6. Pick a task, add a note if you like, and confirm.

Did the progress bar on the widget update? Did you get a notification? Try opening the app. Did the points update there too?

5. Explore settings

  1. Tap the Settings gear icon.
  2. Try changing the colour theme. Does the whole app update instantly?
  3. Change your nickname under Appearance. Does the home screen greeting update?
  4. Try the Export option under Data & Backup. Does it let you save a file?
  5. If you exported, try Import and load the same file back.

6. A few extra things to try

These don't have to be in order, just poke around:

  • Create a reward with no tasks at all and see what you can do with it
  • Create two tasks and mark one mandatory, one optional. Can you claim after only logging the mandatory one (if you have enough points)?
  • Try drag-to-reorder on the Tasks screen: press and hold a task, then drag it up or down.
  • Switch to group view on the Tasks screen (the icon in the top right). Does it look right?
  • Try creating a task with the same name as an existing one.
  • Try creating a reward with the same name as an existing one.
  • Open the Task Library (Tasks screen → Library icon) and import a template.

Thanks again. Feedback at any level of detail is helpful. Even "this felt weird but I can't explain why" is worth sharing. Reach us at hello@secondmondaystudios.com.
